<Aside type="note">
    While the Flatpak is convenient, it comes with some limitations such as dysfunctional Rich Presence. We recommend
    using native packages for the best experience.
</Aside>

[![Download on Flathub](https://dl.flathub.org/assets/badges/flathub-badge-en.svg)](https://flathub.org/apps/org.equicord.equibop)

<IconHeader tag="h4" iconUrl={SteamIcon.src}>
    SteamOS
</IconHeader>

Open the Discover app, search for Equibop and install it. To run it in game mode, add it as a non-steam game.

<IconHeader tag="h4" iconUrl={UbuntuIcon.src}>
    deb (Debian / Ubuntu)
</IconHeader>

Download the right deb package and double click it to install.

<LinkButton href="https://github.com/Equicord/Equibop/releases/latest" data-filename="equibop_VERSION_amd64.deb">x64 Download</LinkButton>
<LinkButton href="https://github.com/Equicord/Equibop/releases/latest" data-filename="equibop_VERSION_arm64.deb">arm64 Download</LinkButton>

<IconHeader tag="h4" iconUrl={FedoraIcon.src}>
    rpm (Fedora / Red Hat)
</IconHeader>

Download the right rpm package and double click it to install.

<LinkButton href="https://github.com/Equicord/Equibop/releases/latest" data-filename="equibop-VERSION.x86_64.rpm">x64 Download</LinkButton>
<LinkButton href="https://github.com/Equicord/Equibop/releases/latest" data-filename="equibop-VERSION.aarch64.rpm">arm64 Download</LinkButton>

<IconHeader tag="h4" iconUrl={ArchIcon.src}>
    AUR (Arch Linux)
</IconHeader>

<LinkButton href="https://aur.archlinux.org/packages/equibop">equibop</LinkButton>
<LinkButton href="https://aur.archlinux.org/packages/equibop-bin">equibop-bin</LinkButton>
<LinkButton href="https://aur.archlinux.org/packages/equibop-git">equibop-git</LinkButton>

<IconHeader tag="h4" iconUrl={AppImageIcon.src}>
    AppImage (Universal)
</IconHeader>

See [this guide](https://discourse.appimage.org/t/how-to-run-an-appimage/80) for how to run an AppImage.

<LinkButton href="https://github.com/Equicord/Equibop/releases/latest" data-filename="Equibop-VERSION.AppImage">x64 Download</LinkButton>
<LinkButton href="https://github.com/Equicord/Equibop/releases/latest" data-filename="Equibop-VERSION-arm64.AppImage">arm64 Download</LinkButton>

<IconHeader tag="h4" iconUrl={AppImageIcon.src}>
    tarball (Portable)
</IconHeader>

To run the portable version, extract it into a folder and run the `equibop` binary inside it. Note that all files are needed, not just the binary!

<LinkButton href="https://github.com/Equicord/Equibop/releases/latest" data-filename="equibop-VERSION.tar.gz">x64 Download</LinkButton>
<LinkButton href="https://github.com/Equicord/Equibop/releases/latest" data-filename="equibop-VERSION-arm64.tar.gz">arm64 Download</LinkButton>
